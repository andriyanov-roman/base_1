Debug - Режим построчного выполнения программы.

Подход к НАСЛЕДОВАНЮ:
 - если у двух классов есть хотябы одно повторяющееся состояние их нужно
 сделать наслениками одного родительского класса.
 - В Java не может быть множественного наследования (имеется ввиду наследоваиние
 сразу нескольких классов одним наследником)

 типы отношений
  - наследование, когда характеристики одного класса передаются другому классу без потребности доп переопр.
      - абстрактный клас это требование к тому как должен уметь(методы) себя вести класс наследник.
      is a == extends
  - агрегация, когда класс "стул" является полем класса "комната"
  - композиция, та же агрегация, только когда оба класса являются полями(состояниями) друг груга.
